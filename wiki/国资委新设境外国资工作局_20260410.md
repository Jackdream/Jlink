# 国资委新设境外国资工作局

## 事件概述

2026年4月8日，国务院国资委正式成立**境外国资工作局**，标志着我国境外资产监管迈入专职化、系统化的新阶段。

## 机构设置

- **局长**：朱凯（曾任国资委规划发展局副局长、国际合作局局长）
- **下设四个业务处**：
  - 国际化经营处
  - 风险防范处
  - 监督治理处
  - 应急管理处

## 主要职责

1. 指导所监管企业国际化经营和境外资产布局优化、结构调整
2. 承担所监管企业境外资产监督有关工作
3. 加强境外投资经营等方面风险防范化解
4. 承担境外突发事件和危机处理有关工作

## 背景数据

| 指标 | 数据 |
|------|------|
| 对外金融资产 | 11.78万亿美元（截至2025年末） |
| 直接投资资产 | 35787亿美元 |
| 对外净资产增长 | 28% |

## 成立意义

- **从"走出去"到"管得住"**：破解境外资产监管三大难题
  - 底数不清
  - 责任不明
  - 手段不足
- **从"被动应对"到"主动防控"**：制度层面补齐短板
- 形成完整监管闭环：事前防范 + 事中监督 + 事后追责

## 与卢麒元建言的关系

卢麒元多年来反复呼吁建立"海外资产管理委员会"：

> "建立海外资产管理委员会，我们离岸人民币太多了，离岸人民币必然形成中国的离岸资产，必须由专门的委员会进行管理。"

此番成立虽名称不同，但核心逻辑一致——对海外资产专职化、系统化监管，是对其建言的积极回应。
