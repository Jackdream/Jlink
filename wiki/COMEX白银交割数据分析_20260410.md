# COMEX白银交割数据全方位分析

> 数据截止：2026年4月10日

---

## 核心数据汇总

### 5月合约交割需求

| 指标 | 数值 |
|------|------|
| 5月主力合约潜在交割需求 | 286.8 M oz |
| 4月10号注册库存 | 76.88 M oz |
| 理论缺口 | 209.9 M oz |
| **缺口比例（无转换）** | **373%** |

### 50%合格库存转换情景

| 指标 | 数值 |
|------|------|
| 实际总可用库存 | 201.51 M oz |
| 50%合格库存用于交割 | 62.32 M oz |

| 缺口 | 147.6 M oz |
| **缺口占5月需求比例** | **29.7%** |

> CME早在2020年承认：合格库存只有50%可用于交割

## 关键时间节点

| 日期 | 事件 |
|------|------|
| **4月28-29日** | 5月合约首个通知日 |
| 剩余工作日 | 12天 |

### 库存转换周期

> **冷知识**：合格库存要转换为注册库存不是一瞬间的事情，大规模的转换（比如100M+ oz）需要**3~7天**。

**关键警示**：
- 如果计划用50%合格库存用于5月交割，需要从此刻开始转换
- 节假日不进行转换操作
- 否则会造成系统拥挤，无法满足交割需求

## 伦敦库存分析

### 库存流向

> 从2026年1月开始，COMEX库存**源源不断流向伦敦**

- 2026年3月伦敦库存：较2月回升，但仍低于12月和1月
- 白银去向：系统外交割，**离开COMEX和伦敦体系**

### 2026年3月伦敦流动性

| 指标 | 数值 |
|------|------|
| 自由白银 | 174.44 Moz |
| 日均清算量 | 200.6 Moz |
| **流动性缺口** | **-26.16 Moz** |

### 流动性预警阈值

| 阈值 | 条件 | 状态 |
|------|------|------|
| 黄线警告 | 自由白银 < 2亿 & 租赁利率 > 3% | - |
| **红线警告** | 自由白银 < 1.5亿 & 租赁利率 > 5% | **未触发** |

> 伦敦当前未达到警告阈值，但ETF浮动可提供一定流动性

## 核心结论

### COMEX当前困境

1. **注册库存严重不足**
   - 5月开仓量巨大（5万多手）
   - 注册库存缺口巨大，完全无法覆盖

2. **时间窗口紧迫**
   - 距离首个通知日仅12天
   - 即使转换50%合格库存，仍有**29.7%缺口**

3. **历史规律**
   - 3月已执行"拔网线"操作
   - 5月可能延续此模式

### 平稳交割的唯一路径

> **减少交割比例**（即降低实际交割量）

### 伦敦现状

- 库存流出持续，不是宽松状态
- 这是COMEX库存持续流出的根本原因
- 流动性缺口为负，但尚未触发警告阈值

## 关键术语

| 术语 | 解释 |
|------|------|
| 注册库存（Registered） | 已检验合格、可立即用于交割的库存 |
| 合格库存（Eligible） | 已申报但尚未完成注册程序的库存 |
| 交割比 | 实际交割量占持仓量的比例 |
| 纸银/实物杠杆 | 纸面合约与实际可用库存的比值 |
| 拔网线 | 交易所通过技术手段限制交易或交割 |
| 自由白银 | 可立即用于清算的白银库存 |
| 租赁利率 | 贵金属租赁市场的利率 |
