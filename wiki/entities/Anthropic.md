# 简介

Anthropic 是由 Dario Amodei 创立的 AI 安全公司，开发 Claude 系列模型和 Mythos 网络安全模型。

# 核心属性

- 类型：公司
- 创始人：[[Dario-Amodei]]
- 主要产品：Claude、Mythos
- 立场：AI 安全优先，拒绝用于大规模监控和自主武器

# 关键要点

- **Project Glasswing**：用前沿 AI 保护全球最关键软件的计划
- **Mythos**：最强网络攻防大模型
- **承诺**：1亿美元使用额度 + 400万美元捐赠
- **与合作方**：AWS、苹果、谷歌、摩根大通、微软、英伟达、Palo Alto Networks、Linux 基金会

# 与美国政府的关系

| 机构 | 关系 |
|------|------|
| **NSA** | 已获准使用 Mythos |
| **白宫** | 积极推进接入 |
| **五角大楼** | 列为"供应链风险"，有诉讼纠纷 |
| **财政部/能源部** | 务实推动 |

# 争议焦点

- **五角大楼要求**：Claude 可用于"所有合法用途"
- **Anthropic 拒绝**：排除大规模国内监控和自主武器开发
- **结果**：合同谈判破裂，2月至今
