# Codex

OpenAI 旗下的核心 AI 编程工具。

## 概述

Codex 是 OpenAI 专注于编程开发的 AI 工具，能够根据自然语言指令生成代码、调试程序、调用开发工具。

## Chronicle 功能

2026年4月21日，Codex 正式推出全新 **Chronicle（编年史）** 功能。

### 核心能力

| 功能 | 说明 |
|------|------|
| 屏幕录制 | 后台静默运行，捕捉代码编写、软件操作、项目调试等全流程动作 |
| 智能梳理 | 内置智能体自动对录制内容进行结构化梳理、核心信息提炼 |
| 本地存储 | 以 Markdown 文件格式保存在用户本地设备 |
| 上下文联动 | Codex 在后续交互中直接调用本地存储的记忆内容 |

### 解决的问题

- 开发者无需反复输入重复信息
- AI 能精准掌握当前项目内容、使用工具、开发阶段及核心需求
- 实现无缝衔接式的编程辅助

### 隐私保护

- 屏幕录制文件仅为临时存储
- 不上传至云端服务器
- 录制完成六小时后自动彻底删除

## 战略意义

Chronicle 功能让 AI 从**被动接收指令**转向**主动感知开发流程**，是 AI 编程工具交互逻辑的革新。
