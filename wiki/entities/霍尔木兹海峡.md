# 霍尔木兹海峡 (Strait of Hormuz)

## 基本信息

| 项目 | 内容 |
|------|------|
| 位置 | 伊朗与阿曼之间 |
| 重要性 | 全球最重要的石油运输通道 |
| 日均石油通过量 | 约2,100万桶（占全球20%） |

## 战略价值

### 全球能源咽喉

- 波斯湾石油出口的唯一海路
- 连接波斯湾与阿曼湾、阿拉伯海
- 封锁将切断全球约20%的石油供应

### 主要依赖国家

| 国家 | 对海峡依赖度 |
|------|-------------|
| 菲律宾 | ~98% |
| 日本 | >70% |
| 韩国 | >60% |
| 印度 | >55% |
| **中国** | **~35%** |
| 美国 | 较低（国内产油+其他来源） |

## 2026年封锁事件

### 时间线

- **2026年4月**：伊朗封锁霍尔木兹海峡
- **封锁时长**：截至报告日期约46天
- **通行量**：暴跌95%
- **油价影响**：一度突破150美元/桶

### 全球影响

- 日韩股市承压
- 欧洲工业界评估成本压力
- 印度央行进入应急状态
- 摩根大通警告库存即将归零

## 中国的应对

### 降低依赖措施

1. **进口多元化**
   - 中俄管道（中亚管道）
   - 西非、美洲海上采购
   - 霍尔木兹依赖度降至35%

2. **替代能源**
   - 煤化工替代
   - 新能源发展
   - 低油气发电占比（3%）

3. **外交布局**
   - 中吉乌铁路（绕开海峡）
   - 中俄能源合作
   - 上合组织框架
