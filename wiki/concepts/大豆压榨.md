# 大豆压榨

将大豆加工成[[豆粕]]和豆油的过程。

## 基本流程

```
大豆 → 清理 → 破碎 → 轧胚 → 浸出 → 蒸炒 → 压榨 → 豆粕 + 豆油
```

## 关键指标

### 压榨量

2026年第15周（4月4日-4月10日）：
- 全国油厂实际压榨量：143.62万吨
- 开机率：39.55%
