# DeepSeek-V4

## 基本信息
- **类型**: 大语言模型
- **开发者**: [[DeepSeek]]
- **架构**: MoE（混合专家模型）
- **状态**: V4-preview已发布（2026-04-24），完整版待发布

## 技术规格

### 模型规模
| 指标 | 数值 |
|------|------|
| 总参数量 | 约1万亿 |
| 激活参数 | 每个token约370亿 |
| 上下文窗口 | 100万token |

### 架构创新
- **Engram条件记忆架构**: 实现超长上下文恒定时间检索
- **MoE架构**: 推理成本与V3持平，效率优先

### 性能指标
| 测试 | 成绩 |
|------|------|
| 100万token信息召回率 | 97% |
| SWE-bench（代码） | 80%+ |
| HumanEval（代码） | 90% |

## 核心能力

### 1. 原生多模态
- 首个支持文本、图像、视频生成的DeepSeek模型
- 此前模型以纯文本为主

### 2. 代码能力强化
- 可处理整个代码仓库级别的复杂bug修复
- 长上下文代码推理超越Claude和GPT系列

### 3. 超长上下文
- 100万token上下文窗口
- 远超V3的128K表现
