# 石油库存

## 定义

石油库存指全球范围内政府储备和商业库存中的原油储备量。

## 当前危机（2026年4月）

### 摩根大通警告
| 指标 | 数据 |
|------|------|
| 封锁时间 | 46天 |
| 1-4月消耗 | 2.5亿桶 |
| 日均消耗 | 约60万桶 |
| **剩余库存** | **接近极限** |

### 即期-期货价差
| 指标 | 数值 |
|------|------|
| 即期价格 | 44美元/桶 |
| 期货价格 | 09美元/桶 |

## 全球库存状态

### 封锁前库存耗尽
- 伊朗封锁霍尔木兹海峡46天
- **4月10日后**：封锁前原油库存彻底耗尽
- **后果**：商业库存强制分配

### 各地影响

| 地区 | 情况 |
|------|------|
| 亚洲 | 印度40%依赖伊朗，已基本停止进口 |
| 欧洲 | 英国50%依赖伊朗，船队面临停摆 |
| 中东 | 伊拉克减产60%，也门燃料危机 |
