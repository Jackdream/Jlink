# 定义

GPT-image-2 是 OpenAI 发布的多模态图像生成模型，在 UI 设计方面能力极强，已接入 Codex 平台并免费使用。

# 核心能力

## UI设计
- **能力极强**: UI设计能力远超GPT-5.5
- **免费使用**: 已接入Codex，免费使用
- **应用场景**: 产品原型、界面设计、视觉设计

## 优势
- 能够理解自然语言指令生成UI设计
- 与GPT-5.5配合效果极佳
- 弥补GPT-5.5在UI设计上的不足

# 最佳实践

## 开发流程
1. **Step 1**: 在 Codex 里使用 GPT-image-2 进行 UI 画图
2. **Step 2**: 用 GPT-5.5 来实现画的图
3. **效果**: 效果很好
