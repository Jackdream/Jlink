# 黄金回家 (Repatriation of Gold)

## 定义

"黄金回家"指各国将存放在海外（主要指伦敦、纽约）的黄金储备运回本国的趋势。

## 主要案例

| 国家 | 行动 |
|------|------|
| 法国 | 2025-2026年出售最后129吨存于纽约的黄金 |
| 德国 | 2013-2017年从纽约巴黎运回674吨 |
| 印度 | 2025年将65%以上黄金储备转回国内 |
| 柬/老/泰 | 从伦敦转移到上海 |

## 驱动因素

### 1. 地缘政治风险
- 俄乌冲突后西方冻结俄罗斯资产的先例
- 对西方金融体系稳定性担忧
- 制裁风险意识增强

### 2. 主权安全
- 存放海外存在被冻结风险
- "自己的黄金自己保管"

### 3. 新兴枢纽崛起
- 上海黄金交易所：2023年交割量全球第一
- 新加坡：政治稳定，税收优惠
- 迪拜：中东最大非主权黄金储存库

## 全球格局变化

```
传统格局（单极）          新兴格局（多极）
伦敦/纽约/苏黎世   →    上海/新加坡/迪拜/本国
```

## 数据支撑

- 伦敦金库：8,865吨（2025年7月）
- 上海黄金交易所：2023年场内交割量3,912吨
- 新加坡"The Reserve"：2025年初至4月存储订单增长88%
