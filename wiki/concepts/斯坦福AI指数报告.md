# 定义

斯坦福大学人类中心人工智能研究所（Stanford HAI）每年发布的《AI Index Report》，是全球AI领域最权威的报告之一，追踪全球AI发展动态和关键指标。

# 核心思想

- 中美AI模型性能差距快速缩小
- 竞争焦点从模型分数转向应用落地
- 中国在专利、基础设施、民心等方面具有优势
- AI研发成本曲线被重写

# 组成部分/子问题

- 模型性能基准测试（MMLU、MATH、HumanEval）
- AI研发投入统计
- AI专利数据分析
- 工业机器人装机量统计
- 人才流动趋势
- 民众对AI的态度调查

# 支持证据

| 指标 | 2023年 | 2026年 | 变化 |
|------|--------|--------|------|
| MMLU差距 | 美国+17.5分 | 接近 | - |
| MATH差距 | 美国+24.3分 | 接近 | - |
| HumanEval差距 | 美国+31.6分 | 接近 | - |
| AI专利占比 | - | 中国69.7% | - |
| 工业机器人 | - | 中国50%全球 | - |

关键引用："2023年，美国模型全面碾压中国。这个趋势，已经不再成立了。"

# 对立观点/张力

- 顶尖模型仍以美国为主（GPT-4、Claude、Gemini）
- 专利数量vs专利质量的讨论
- 模型分数"够用"后的实际价值
