# 去工业化

发达国家制造业向海外转移、工业规模持续萎缩的现象。

## 核心定义

去工业化是指一个国家的工业（尤其是制造业）在经济结构中占比下降，服务业占比上升的过程。

## 典型案例

### 日本

| 指标 | 数据 |
|------|------|
| 2024年发电量 | 9819.93 亿度 |
| 历史峰值 | 1.002648 万亿度（2010年） |
| **下降幅度** | **-16.13%** |

### 德国

| 指标 | 数据 |
|------|------|
| 2023年发电量 | 5118.81 亿度 |
| 历史峰值 | 2017年 |
| **下降幅度** | **-21.7%** |

### 美国

| 指标 | 数据 |
|------|------|
| 工业用电占比 | 25.68%（2025年） |
| 居民用电占比 | 37.33% |
| 工业用电增速 | +0.7%（缓慢） |

## 去工业化后果

### 1. 工业能力下降

- 制造业空心化
- 产业链不完整
- 关键物资依赖进口

### 2. 军工产能受限

从俄乌战争看西方军工产能：

| 物资 | 情况 |
|------|------|
| 大口径炮弹 | 四年期间始终落后于俄罗斯 |
| 无人机 | 主要靠乌克兰自产 |
| 远程武器 | 依赖伊朗进口（沙赫德） |

### 3. 就业结构变化

- 制造业就业减少
- 服务业就业增加
- 中产阶级萎缩

## 反驳"工业4.0"论

> 哪有发电量下降这么多的工业4.0

德国提出的工业4.0转型，但实际发电量持续下降，制造业外迁趋势明显。

## 对中国的启示

| 优势 | 说明 |
|------|------|
| 工业规模 | 是美国的6倍、日本的18倍、德国的29倍 |
| 增速 | 工业用电年增3.7%，远超西方 |
| 产业链 | 完整工业体系 |
| 能源 | 电力供应充足 |
