# Project Glasswing（玻璃翼计划）

**全称**: Glasswing Project

**发起方**: Anthropic

**发布时间**: 2026年4月8日

## 背景

为应对Claude Mythos带来的网络安全威胁，Anthropic联合行业巨头发起的一项安全防御计划。

## 参与方

- Amazon
- Apple
- Google
- Linux Foundation
- Microsoft
- NVIDIA
- 40+关键软件基础设施的建设者和维护者

## 核心措施

1. **防御性安全研究**: 合作方将Mythos Preview投入自身防御性安全工作
2. **提前解锁**: 允许合作方利用Mythos能力加强防御
3. **开源安全支持**: 扫描和加固自研系统及开源系统
4. **资金支持**: 提供最高1亿美元使用额度
5. **捐赠**: 向开源安全组织捐赠400万美元

## 核心理念

> Anthropic认为：网络安全没有任何一家机构可以单独解决，前沿AI公司、软件厂商、安全研究员、开源社区、各国政府都得一起参与。

## 目标

在Claude Mythos正式版发布前，尽可能加固全球网络基础设施，应对AI能力跃迁带来的安全挑战。
