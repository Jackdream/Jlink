# 六张网

## 定义
国家发改委推出的六大基础设施网络体系，总投资约7万亿元，构成"十五五"全域产业发展底盘。

## 六大网络

| 网络 | 核心方向 | 落地区域 |
|------|----------|----------|
| 国家水网 | 跨流域调水、防洪抗旱、城乡供水 | 长江、黄淮海、西北 |
| 新型电力网 | 特高压、储能、新能源并网 | 西北光伏、华北风电、中东部负荷区 |
| 全国算力网 | 东数西算、智算中心、算力调度 | 贵州、甘肃、内蒙古、宁夏 |
| 新一代通信网 | 5G/6G、卫星互联网、空天地海 | 全国全域+海洋空域 |
| 城市地下管网 | 管网改造、综合管廊、雨污分流 | 一二线+强三线老城 |
| 全国物流网 | 多式联运、物流枢纽、冷链降本 | 枢纽城市群、沿江沿海 |

## 资金结构（~7万亿）

| 来源 | 金额 |
|------|------|
| 中央预算 | 7,550亿 |
| 超长期国债 | 10,000亿 |
| 政策性金融 | 8,000亿 |
| 地方专项债 | 44,000亿 |

## 战略逻辑

### 四层逻辑

1. **不注水金融泡沫**: 全部投向实体固定资产，与西方量化宽松形成对比
2. **对内降成本**: 降低生产生活成本，稳就业托底层
3. **内循环底座**: 搭建完整内循环，剥离美元环流绑定
4. **十五五底盘**: 锁死长期经济安全+新产能

### 与已有框架的关系

- [[新发展格局]]: 六张网是内循环的物理底座
- [[能源安全]]: 新型电力网直接服务于能源自主
- [[科技自立自强]]: 算力网+通信网是科技自主的数字底座
- [[东升西降]]: 7万亿投实体 vs 美国金融泡沫的结构性差异
- [[海洋经济]]: 通信网空天地海一体化覆盖海洋空间

## 产业链传导

```
中央财政 + 超长期国债 + 地方专项债
            ↓
六张网基建投资（水利/电力/算力/通信/管网/物流）
            ↓
上游材料设备需求 → 中游工程建设 → 下游运营服务
            ↓
降低全社会生产生活成本 → 增厚制造业利润 → 支撑内循环
```
