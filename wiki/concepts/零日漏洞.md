# 定义

零日漏洞（Zero-day）是指软件或硬件中被发现但尚未修补的安全漏洞，"零日"指被发现时尚无补丁可用。

# 核心特点

| 特点 | 说明 |
|------|------|
| **发现难度** | 需要高水平的安全研究 |
| **危险性** | 在修复前可被利用 |
| **价值** | 在黑市和国家级攻击中高价交易 |

# Mythos 的突破

- Anthropic 的 Mythos 模型已在关键基础设施中识别出**数千个**零日漏洞
- 这改变了漏洞发现和修复的节奏
- 使得网络安全防御更加主动

# 与 AI 的关系

**双刃剑效应**：
- **防守方**：AI 帮助更早识别和缓解风险（AWS、微软等公司的反馈）
- **攻击方**：AI 也会让攻击者更快发现和利用漏洞（Palo Alto Networks 的警示）
