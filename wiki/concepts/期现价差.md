# 期现价差

期货价格与现货价格之间的差值。

## 基本公式

```
期现价差 = 现货价格 - 期货价格
```

## 市场含义

| 状态 | 含义 |
|------|------|
| 正值（升水） | 现货价格高于期货 |
| 负值（贴水） | 现货价格低于期货 |

## 数据（2026年4月16日）

| 区域 | 期现价差（元/吨） |
|------|-----------------:|
| 天津 | -1 |
| 山东 | -101 |
| 江苏 | -111 |
| 广东 | -91 |
| **沿海均价** | **-111至-1** |

## 影响因素

- 现货供需紧张程度
- 仓单注册量
- 交割成本
- 市场预期

## 分析意义

本周各区域期现价差均值较节前一周大幅下调，主要因：
- 现货供应宽松预期
- 下游采购意愿谨慎
- 油厂库存维持高位
