# 总体国家安全观

## 定义
习近平总书记从新时代坚持和发展中国特色社会主义的战略高度，创造性提出的国家安全理念，强调"要统筹发展和安全""加快推进国家安全体系和能力现代化，以新安全格局保障新发展格局"。

## 核心要义

### 统筹发展和安全
- 发展是"第一要务"
- 安全是"头等大事"
- 以新安全格局保障新发展格局

### 重点领域
| 领域 | 重点内容 |
|------|----------|
| 粮食安全 | 藏粮于地、藏粮于技，确保谷物基本自给、口粮绝对安全 |
| 能源资源安全 | 四个革命、一个合作，能源自给率保持80%以上 |
| 产业链供应链安全 | 关键时刻不能掉链子，制造业全球领先 |
| 科技安全 | 高水平科技自立自强，突破"卡脖子"技术 |
| 金融安全 | 防范化解房地产、地方政府债务、中小金融机构风险 |
| 数据安全 | 强化数据分类分级保护，规范数据跨境流动 |
| 人工智能安全 | 完善法律法规、伦理准则，推动安全能力建设 |

## 经济安全保障能力建设

### 过去五年成效
- 经济总量迈上140万亿元新台阶
- 粮食产量连续两年站稳1.4万亿斤
- 能源自给率保持在80%以上
- 制造业增加值连续16年全球第一

### "十五五"重点任务
1. 立足国家经济安全扎实办好自己的事
2. 提升经济安全韧性和抗冲击能力
3. 完善经济安全风险防控体系
4. 加强新兴领域安全治理

- [[新质生产力]] - 推动高质量发展的新动能
- [[科技自立自强]] - 关键核心技术自主可控
- [[产业链供应链安全]] - 确保关键时刻不掉链子
