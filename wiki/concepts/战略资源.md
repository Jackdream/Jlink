# 定义

战略资源是指在特定历史时期对国家或部落生存、发展具有决定性作用的资源，其分布会影响政治格局和文明走向。

# 上古时期的战略资源

## 盐

| 特点 | 说明 |
|------|------|
| **重要性** | 与粮食同等重要 |
| **来源** | 海盐需到海边获取 |
| **分布** | 山东等沿海地区 |

## 铜

| 特点 | 说明 |
|------|------|
| **双重价值** | 货币 + 青铜器 |
| **军事价值** | 兵器 |
| **生产价值** | 农具、其他生产工具 |

# 地缘政治分析

## 山西（尧的势力范围）

- 鱼米之乡，适合人类生存
- 中原腹地，中华文明发源地
- **缺陷**：缺乏盐、铜

## 山东（东夷，舜的势力范围）

- 海盐资源丰富
- 铜矿资源
- **优势**：战略资源充沛

# 对历史的影响

```
盐+铜在东夷
    ↓
尧需要向东发展
    ↓
发现并培养舜
    ↓
部落合并（禅让）
    ↓
中华文明核心区形成
```

# 核心洞见

> "四千五百年前，有两样东西变得越来越重要：盐和铜。"

- 战略资源分布决定了上古部落扩张方向
- 资源的互补性推动了部落联合
- 理解战略资源是理解上古历史的关键
