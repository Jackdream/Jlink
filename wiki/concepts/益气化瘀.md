# 益气化瘀

## 基本概念
- **属性**: 中医治法
- **组成**: 益气法 + 化瘀法
- **适用**: 气虚血瘀证

## 理论依据

### 气与血的关系
> **气为血之帅，血为气之母**

- **气能生血**: 气的运化是血液生成的动力
- **气能行血**: 气的推动是血液运行的动力
- **气能摄血**: 气的固摄控制血液在脉中运行
- **血能载气**: 血是气的载体

### 气虚血瘀的病机
- 气虚推动无力 → 血行不畅 → 瘀血内停
- 气虚固摄无权 → 血溢脉外 → 瘀血形成
- 瘀血阻滞 → 气机不畅 → 加重气虚

## 治法特点

### 单纯益气 vs 益气化瘀
| 治法 | 适用 | 局限 |
|------|------|------|
| 单纯益气 | 单纯气虚 | 瘀血难消 |
| 单纯化瘀 | 单纯血瘀 | 正气更伤 |
| **益气化瘀** | **气虚血瘀** | **扶正祛邪，标本兼治** |

## 朱良春应用

### 在慢性肾炎中的应用

#### 病机认识
- 脾肾两虚 → 气虚
- 病程日久 → 入络成瘀
- **气虚血瘀**是慢性肾炎的核心病机之一

#### 代表方剂
**益气化瘀补肾汤**

| 药物 | 用量 | 功效 |
|------|------|------|
| 生黄芪 | 30g | 益气培本 |
| 全当归 | 10g | 养血活血 |
| 川芎 | 10g | 行气活血 |
| 红花 | 10g | 活血化瘀 |
| 丹参 | 30g | 活血祛瘀 |
| 仙灵脾 | 15g | 温肾助阳 |
| 川续断 | 10g | 补肝肾 |
| 怀牛膝 | 10g | 活血下行 |
| 石韦 | 20g | 利水通淋 |
| 益母草 | 120g | 活血利水（煎汤代水） |

#### 疗效特点
- 治疗隐匿型肾炎10例
- 4例完全缓解
- 5例基本缓解
- 1例部分缓解
- **全部有效**

### 用药特色

#### 重用益母草
- **用量**: 120g（煎汤代水煎药）
- **功效**: 活血利水
- **特点**: 药专力宏，屡用得效

#### 黄芪的用法
- **常规**: 30g
- **气虚甚**: 加太子参30g
- **稳定期**: 重用至90g

|------|------|

| 益气化瘀 + 养阴 | 阴虚血瘀 |

| 益气化瘀 + 解毒 | 气虚血瘀毒蕴 |

## 注意事项

### 使用原则
1. **辨证准确**: 必须是气虚血瘀证
2. **分清主次**: 气虚为主还是血瘀为主
3. **掌握分寸**: 益气与化瘀的比例
4. **顾护脾胃**: 久病注意脾胃功能

### 禁忌
- 单纯气虚无明显瘀血者，不宜过用化瘀
- 单纯血瘀正气未虚者，不宜过用益气
- 出血倾向明显者，化瘀药慎用
