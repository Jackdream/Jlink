# 时序抽象 (Temporal Abstraction)

## 定义

时序抽象是分层强化学习中的一种核心思想，指的是在不同时间尺度上进行决策和控制。

## 在分层强化学习中的作用

- **上层控制器**：在较长时间跨度上选择option/goal
- **下层控制器**：在较短时间跨度上根据option/goal选择action

## 与Option的关系

Sutton最早在论文《Between MDPs and semi-MDPs》中提出：
- **半马尔科夫过程（Semi-MDP）**
- **Option框架**

Option表示的是一种具有时序抽象特性的策略，是上层策略的抽象表达。

## 核心思想

```
长时间跨度（上层）
┌──────────────────────────────────┐
│ 选择 Option/Goal                 │
│  ↓ 内部奖励传递                  │
│ 监督下层策略好坏                  │
└──────────────────────────────────┘
         ↓
短时间跨度（下层）
┌──────────────────────────────────┐
│ 根据 Option/Goal 选择 Action      │
│ 执行具体动作                      │
└──────────────────────────────────┘
```

## 优势

1. **分解复杂任务**：将长期目标分解为短期动作
2. **提升样本效率**：内部奖励加速学习
3. **迁移泛化**：子策略可在不同任务间复用
