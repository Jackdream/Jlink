## 核心病机

慢性肾炎的致病因素比较复杂：
- **内在因素**: 脾肾两虚
- **发病诱因**: 风寒湿热
- **病理基础**: 脏腑、气血、三焦气化功能失调

**治疗原则**: 标本兼顾，补泄并施，益气化瘀，通腑泄浊

---

## 7点临床经验

### 1. 温补脾肾为要法

**主要证型**: 脾肾阳虚

**关键药物**:
| 药物 | 功效 |
|------|------|
| **附子** | 温肾，有肾上腺皮质激素样作用 |
| **仙灵脾** | 温肾，有肾上腺皮质激素样作用 |
| **黄芪** | 益气培本，促进血循，利水 |

**禁忌**: 舌质红绛、湿热炽盛者慎用

---

### 2. 益气化瘀补肾汤治隐匿型肾炎

**疗效观察**: 10例中4例完全缓解，5例基本缓解，1例部分缓解，全部有效

**方剂组成**:
- 生黄芪30g
- 全当归、川芎、红花各10g
- 丹参30g
- 仙灵脾15g
- 川续断、怀牛膝各10g
- 石韦20g
- **益母草120g**（煎汤代水煎药）

**加减法**:

| 证型 | 加减 |
|------|------|
| 急性发作/严重蛋白尿 | 去黄芪、红花，加金银花、连翘、漏芦、菝葜各15g，地鳖虫10g，鱼腥草、白花蛇舌草各30g，蝉衣5g |
| 肾功能低下为主 | 加炮山甲8g |
| 阳虚 | 加附子、肉桂、鹿角霜、巴戟天 |
| 肾阴虚 | 加生地黄、龟版、枸杞子、女贞子、旱莲草 |
| 脾虚 | 加党参、白术、山药、苡仁 |
| 气虚甚 | 重用黄芪，加太子参30g |
| 肾关不固 | 加金樱子、芡实、益智仁 |
| 浮肿明显伴高血压 | 加水蛭2g（研末，胶囊装，分吞） |
| 血尿 | 加琥珀3g（研，分吞），茅根30g |
| 血压高 | 去川芎，加桑寄生30g，广地龙15g |

---

### 3. 水肿消除多法并举

**常用药物**: 生黄芪、制附子、石韦

**特效药物**: 益母草大量（有明显的活血利水作用）

**尿少短涩**: 蟋蟀20g、沉香5g，共研细末，胶囊装，每服6粒，每日2～3次

---

### 4. 蛋白尿消退难点的处理

**病机**: 肾之闭藏失职，精气外泄，阳不摄阴

**特效药物**:
| 药物 | 用量 | 功效 |
|------|------|------|
| **石韦** | 30～60g | 消除肾小球病变，抑制免疫反应 |
| **仙鹤草** | - | 消除尿蛋白 |
| **益母草** | - | 消除尿蛋白 |
| **生槐米** | 45g | 配合土茯苓45g、菝葜30g |

**兼症处理**:
- 血胆固醇高：加强运脾之品
- 颗粒/透明管型多：加强滋肾补肾（山萸肉、枸杞子）

---

### 5. 慢性肾功能衰竭与尿毒症

**病机**: 肾虚为本，湿热、水毒、浊瘀为标

**关键指标**: 血尿素氮和肌酐

**治疗原则**: 温肾补肾 + 化湿热、利水毒、泄浊瘀

#### 中药保留灌肠（中医肠道透析法）

**适应证**: 尿素氮和肌酐持续升高，浊阴上干，频繁呕吐，服药困难

**灌肠方**:
- 生大黄10～20g
- 白花蛇舌草、六月雪各30g
- 丹参20g

**加减**:
| 证型 | 加减 |
|------|------|
| 阴凝征象 | 加熟附子15g、苍术20g |
| 血压高/出血倾向 | 加生槐米45g、广地龙15g |
| 湿热明显 | 加生黄柏20g |
| 阴虚 | 加生地黄、川石斛各20g |

**用法**: 合方煎成200ml，每日1～2次，保留灌肠

#### 配合治疗

**醒脑静注射液**: 每次2～4支，加10%葡萄糖40ml，缓缓推注，每6小时一次

**温肾解毒方**:
- 熟附子10～20g
- 生白术20g
- 姜半夏10g
- 紫丹参、六月雪、扦扦活各30g
- 党参15g
- 绿豆、白花蛇舌草、半枝莲各30g
- 黄连2g
- 益母草120g（煎汤代水煎药）

**加减**:
- 肌酐和尿素氮不下降：加白金丸6g（包煎）
- 皮肤瘙痒：加白鲜皮、地肤子各30g
- 血压高/出血倾向：加生槐米45g，广地龙15g

**稳定后**: 重用黄芪90g，仙灵脾30g，温肾助阳，益气利水

**尿量少**: 蟋蟀10g，人工牛黄1g，琥珀4g，共研细末，胶囊装，每服4粒，每日2次

---

### 6. 舌体形态预测预后

| 舌体形态 | 预后 |
|----------|------|
| **瘦长而薄** | 险恶 |
| **胖大** | 较佳 |

**理论依据**: 舌为心之苗，心与肾均属少阴经，足少阴肾经络舌本

---

### 7. 巩固治疗与康复调养

#### 巩固治疗
- **病情稳定后**: 长期服用丸剂
- **偏阴虚**: 六味地黄丸
- **偏阳虚**: 金匮肾气丸

#### 冬虫夏草
- **功效**: 巩固疗效，改善肾功能，提高细胞免疫功能，降低尿素氮和肌酐
- **用法**: 每日1g煎汤连渣服，或研末胶囊每日4粒
- **缺点**: 价格昂贵，货源紧张

#### 康复期注意事项

**生活调摄**:
- 生活多样化、节律化
- 静中寓动，适度户外活动
- 适应时令变化
- 避免呼吸道感染

**饮食宜忌**:
- 以清补为主
- 不宜辛辣刺激
- 不宜含盐分过高

## 核心用药特点

### 重用药物
| 药物 | 用量 | 用途 |
|------|------|------|
| 益母草 | 120g | 煎汤代水，活血利水 |
| 黄芪 | 30～90g | 益气培本 |
| 石韦 | 30～60g | 消除蛋白尿 |
| 生槐米 | 45g | 配合治疗 |

### 特色制剂
- 蟋蟀+沉香：利尿胶囊
- 水蛭：化瘀利水（研末胶囊）
- 琥珀：止血（研末分吞）
- 白金丸：降肌酐尿素氮
