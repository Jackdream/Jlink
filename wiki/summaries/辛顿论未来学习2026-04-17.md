# 辛顿：未来这样学，做得更远

## 基本信息

- 来源：辛顿采访视频整理
- 类型：AI与教育观点

## 核心观点

### AI替代人类的共识

- 未来AI会在很多领域替代人
- 现阶段及一段时间内，AI与人仍是协作关系
- AI终将替代大量人类工作
- 很多岗位不再需要人来做

### 未来学习方向

辛顿给出了两个方向：

#### 方向一：具备实操手艺

| 特点 | 说明 |
|------|------|
| 类型 | 管道工、水管维修工等 |
| 依赖 | 手工技能 |
| 优势 | 在很长一段时间内具有生存优势 |

#### 方向二：扎实通识基础 + STEAM素养

| 要求 | 说明 |
|------|------|
| 了解 | 古希腊哲学 |
| 看懂 | 现实 |
| 掌握 | 数学、物理、科学等 |

### 核心结论

> **最重要的是，要学会思考。会思考的人，未来能走得更远。**

## 观点总结

| 类型 | 核心要点 |
|------|----------|
| AI影响 | 替代大量人类工作 |
| 协作期 | 现阶段AI与人协作 |
| 生存技能1 | 实操手艺（管道工等） |
| 生存技能2 | 通识基础 + STEAM素养 |
| 最重要 | 学会思考 |
