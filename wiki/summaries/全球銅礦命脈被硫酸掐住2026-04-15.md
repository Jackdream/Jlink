# 全球铜矿命脉被硫酸掐住

## 核心论点

全球铜供应链正从"资源约束"切换到"化学约束"。过去坚如磐石的链条——矿山开采、精矿冶炼、电解铜——正被新链条取代：**硫磺→硫酸→湿法铜→全球供应**。

## 两大冲击因素

### 1. 东方"关闸"：中国限制硫酸出口

| 数据 | 说明 |
|------|------|
| 2025年中国硫酸出口 | 约460万吨 |
| 其中流向智利 | 约30%（超100万吨） |
| 智利依赖程度 | 维持其1/5铜产量 |
| 2026年前两月 | 对智利出口已萎缩一半 |
| 政策 | 2026年5月起限制硫酸出口 |

**原因**：中国国内新能源电池、磷肥、高端制造对硫酸需求膨胀

### 2. 中东"断源"：霍尔木兹海峡危机

| 数据 | 说明 |
|------|------|
| 中东硫磺产量 | 占全球1/4 |
| 海运硫磺过霍尔木兹 | 占全球1/2 |
| 硫磺价格涨幅 | 4750元→6750元/吨 |

**影响**：伊朗冲突升级导致船舶滞留、战争保险费率飙升

## 产业链的关键变化

### 利润表倒置

- 铜精矿加工费已历史性进入负值区间
- 硫酸贡献了**超过60%的非主业利润**
- 冶炼厂实质已变成"制酸厂"

### 湿法铜（SX-EW工艺）困境

- 不需要选矿和冶炼炉，只需把硫酸洒在矿石堆上
- 成本低、投产快，是铜价上涨时最敏捷的响应者
- **但现在被困：没有酸=没有铜**

## 核心观点

> **过去我们问：哪里还有矿？现在我们必须问：哪里还有酸？**

> **谁手里握着那瓶决定产量的无色液体，谁才真正握着铜世界的权杖。**

## 受影响最大的地区

| 地区 | 情况 |
|------|------|
| **智利** | 最大铜生产国，五分之一铜产量依赖进口硫酸 |
| **刚果（金）** | 氧化铜矿对湿法冶金依赖极深 |
| **赞比亚** | 2025年9月已出台政策限制硫酸出口保障本土供应 |

## 硫酸供应链链条

```
波斯湾炼油厂脱硫 → 硫磺运往中国 → 制成硫酸
                                    ↓
智利矿山堆浸 ← 湿法铜 ← 硫酸出口
```
