# 核心事件

**美国国安局（NSA）已在使用 Anthropic 最强模型 Mythos，而五角大楼却将其列为"供应链风险"。**

## 主要人物

| 人物 | 职位 |
|------|------|
| Dario Amodei | Anthropic CEO |
| Susie Wiles | 白宫幕僚长 |
| Scott Bessent | 财政部长 |
| Jerome Powell | 美联储主席 |
| Gregory Barbaccia | 联邦首席信息官 |

# 核心事件时间线

## 4月7日：Project Glasswing 发布
Anthropic 推出 Project Glasswing，定义是"用前沿 AI 保护全球最关键的软件"。

**Mythos 合作方**：
- AWS、苹果、谷歌、摩根大通、微软、英伟达
- Palo Alto Networks、Linux 基金会
- 另有 40+ 组织获得有限访问权限

**承诺**：1亿美元使用额度 + 400万美元捐赠

## 4月上旬：金融系统紧急会议
- 财政部长 Scott Bessent 与美联储主席 Jerome Powell 召集华尔街银行高管
- 讨论 Mythos 带来的网络安全风险
- 监管层视为"系统性风险议题"

## 4月中旬：白宫推进
- 联邦首席信息官 Gregory Barbaccia 向内阁部门发信
- 要求技术与安全负责人为 Mythos 接入做准备
- 通知范围：国防部、财政部、商务部、国土安全部、司法部、国务部

## 4月19日：NSA 使用曝光
- Axios 披露 NSA 已在使用 Mythos Preview
- 使用范围可能涉及漏洞发现、防御性网络安全、系统加固

# 五角大楼 vs NSA 的分歧

| 机构 | 态度 |
|------|------|
| **五角大楼** | 将 Anthropic 列为"供应链风险"，与之有诉讼纠纷 |
| **NSA** | 已在使用 Mythos，且使用范围在扩大 |
| **能源部/财政部** | 务实态度，推动接入 |

## 争议焦点

- **五角大楼要求**：Claude 模型可用于"所有合法用途"
- **Anthropic 拒绝**：排除大规模国内监控和自主武器开发
- **结果**：2月合同谈判破裂，争议延续至今

# 白宫的策略

**"修改版" Mythos** 的两层含义：

1. **技术层面**：定义哪些能力可调用、哪些行为被拦截、日志留痕、输出审查
2. **制度层面**：限定在防御性网络安全、漏洞挖掘、系统加固用途

**推进路径**：先放进制度，再放进机构

# 关键风险

证券业协会警告的风险：
- 散户数据泄露
- 交易策略外泄
- 被外国行为体利用
- 金融系统连锁失灵

# 关键洞见

1. **模型能力转变**：Mythos 开始改变漏洞发现和修复的节奏
2. **政府内部判断分层**：一部分人认为价值观限制是风险，另一部分认为网络安全需求更紧迫
3. **监管转向**：从"算力、芯片、出口管制"转向"部署安全"
4. **NSA 已用**：说明美国政府内部的选择其实已经发生
