# 概要

毛泽东1917年24岁时写成《心之力》，被誉为"建国之才的奇文"，曾被湖南一师杨济昌老师打满分105分。此文体现了青年毛泽东的爱国思想、哲学观点和救国抱负，被视为毛泽东思想的坚实基础。

- **核心思想**: 心力决定一切，强调精神力量的重要性

# 核心论点

## 1. 心力为本
> "天之力莫大于日，地之力莫大于电，人之力莫大于心"

- 心为万力之本，由内向外可生善、可生恶、可创造、可破坏
- 修之以正则可造化众生，修之以邪则能涂炭生灵

## 2. 救国之道
- 首要谋划：自强国民心力
- 破除封建、官僚愚昧邪道
- 惩治卖国、汉奸、洋买办
- 以国家民族之新生心力志向缔造世界仁德勇武文明之新学

## 3. 四贼论
自中国开埠以来，教会、资本家、君主、卖国贼四者同为天下恶魔强盗，四贼之中官僚最为紧要

## 4. 青年使命
> "故吾辈任重而道远，若能立此大心，聚爱成行，则此荧荧之光必点通天之亮，星星之火必成燎原之势"

# 重要引用

- "宇宙即我心，我心即宇宙"
- "世界、宇宙乃至万物皆为思维心力所驱使"
- "为天地立心何以立？为生民立命何以立？为往圣继绝学何以继？为万世开太平何以开？"
- "天下兴亡匹夫有责"
- "与时俱进"（毛泽东近100年前率先使用）

# 解读/意义

1. **思想基础**: 《心之力》被视为毛泽东思想的坚实而全面的基础
2. **预言性质**: 新中国的成就早在此文中预先展示
3. **历史价值**: 一个民国青年有如此爱国宏志，至今仍为世人楷模
4. **现实意义**: 以今天的国际和国内局势来看，此文毫不过时
