# 核心数据

## 蒙古国石油进口

| 指标 | 数据 |
|------|------|
| 年度进口量 | 200-270万吨 |
| 来源：俄罗斯 | 80%-95% |
| 来源：中国 | 5%-20% |
| 2026年预测 | 100%从俄罗斯进口 |

## 俄罗斯石油出口影响

| 指标 | 数据 |
|------|------|
| 俄罗斯石油出口能力暂停 | ≥40% |
| 影响范围 | 蒙古国矿山开采、煤炭运输 |

| 库存位置 | 数量 |

| 查干哈达堆煤场 | 110万吨 |

| 指标 | 数据 |
|------|------|

# 关键结论

1. **短期无虞**：ETT矿山库存充足（430万吨），可应对短期燃料短缺
2. **进口高位**：4月份蒙煤进口量预计依旧高位
3. **风险存在**：俄罗斯石油出口受阻，长期存在减量预期
4. **预案启动**：ETT矿山已于4月中旬召开董事会制定预案

---

# 背景分析

## 蒙古国能源结构

蒙古国石油产品完全依赖进口，高度依赖俄罗斯。2026年起将100%从俄罗斯进口，风险高度集中。

## 对中国的影响

- 甘其毛都是中蒙重要的煤炭进口口岸
- 蒙煤是中国煤炭进口的重要来源
- 燃料短缺暂未影响进口，但需关注后续演变
