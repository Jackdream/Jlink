# 中国四大跨国能源进口通道

## 核心价值

**减少对马六甲海峡的依赖**，构建多元化、多方向的能源供应网络。

---

## 四大通道

### 1. 中俄输油管道

| 指标 | 数据 |
|------|------|
| 起点 | 俄罗斯斯科沃罗季诺 |
| 终点 | 中国大庆 |
| 长度 | 999.04公里（中国境内927公里） |
| 年输油能力 | 1,500万吨 |
| 运营时间 | 2011年1月1日 |
| 合同期 | 20年 |

**战略意义**：
- "贷款换石油"协议成果
- 保障东北地区能源供应
- 深化中俄能源合作

---

### 2. 中哈输油管道

| 指标 | 数据 |
|------|------|
| 起点 | 哈萨克斯坦阿特劳 |
| 终点 | 新疆阿拉山口 |
| 长度 | 2,798公里（"丝绸之路第一管道"） |
| 运营时间 | 一期2006年，二期2009年 |

**战略意义**：
- 保障独山子、兰州等西部炼厂
- "一带一路"能源合作典范
- 带动哈萨克斯坦经济发展

---

### 3. 中巴输油管道（在建）

| 指标 | 数据 |
|------|------|
| 起点 | 巴基斯坦瓜达尔港 |
| 终点 | 新疆喀什 |
| 状态 | 在建 |

**战略意义**：
- 绕开马六甲海峡
- 直接从印度洋获取能源
- 极大提升能源运输安全性

---

### 4. 中缅输油管道

| 指标 | 数据 |
|------|------|
| 原油管道起点 | 缅甸马德岛 |
| 天然气管道起点 | 皎漂港 |
| 入境点 | 云南瑞丽 |
| 原油管道长度 | 771公里 |
| 天然气管道长度 | 793公里 |
| 投产时间 | 天然气2013年，原油2015年 |

**战略意义**：
- 第四大能源进口通道
- 原油运输无需经过马六甲海峡
- 覆盖西南地区能源需求

---

## 战略价值矩阵

## 分析意义

1. **能源自主性**：四大通道降低单一路线风险
2. **地缘布局**："一带一路"基础设施投资成果
3. **应对危机**：如霍尔木兹封锁，可通过其他通道保障供应
