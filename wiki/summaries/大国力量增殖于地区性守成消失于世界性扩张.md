# 大国力量增殖于地区性守成消失于世界性扩张

## 核心命题

> **"大国力量增殖于地区性守成，消失于世界性扩张。"**

战略的核心不是力量投射，而是对**力量边界**的认识。

## 关键要点

### 1. 战略定义
- 战略是"刀尖上的哲学"
- 关键在于认识力量边界，而非单纯投射力量

### 2. 核心原则
- 大国应在自身能力边界内维持地区性守成
- 不应进行不可消化的世界性扩张
- 近代日本军费占 GDP 比例不断升高的案例说明扩张与国家可持续性之间的张力

### 3. 中国的地缘优势
- 兰州及河西走廊
- 东北工业与人口基础
- 大西南的守成与反攻空间

### 4. 国际比较
| 国家 | 特点 |
|------|------|
| 日本 | 扩张导致可持续性崩溃 |
| 美国 | 世界性扩张的代价 |
| 俄罗斯 | 版图约束与安全半径 |
| 中国 | 独特的地缘结构优势 |

### 5. 政教关系
- "两个中心曰患"：解释欧洲碎片化与中国长期统一能力的重要线索

### 6. 国家战略能力
- 宗教-政治关系
- 学风
- 教育与人才布局
