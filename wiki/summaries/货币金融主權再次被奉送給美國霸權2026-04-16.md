# 货币金融主权，再次被奉送给美国霸权

——评香港稳定币牌照发给汇丰、渣打

## 核心事件

- **时间**：2026年4月10日
- **事件**：香港金融管理局根据《稳定币条例》向两家机构授予稳定币发行人牌照
  - 碇点金融科技有限公司（渣打银行占50%股份）
  - 香港上海汇丰银行有限公司
- **牌照生效**：当日生效，计划于2026年下半年推出港元或美元挂钩的合规稳定币

## 作者的主要观点

### 1. 对稳定币牌照的定性

作者认为这标志着：
- "美元稳定币"骗局在香港得逞
- 香港金融当局充当"美元稳定币"骗局的帮凶
- 香港在货币主权、金融主权上仍属于美国霸权

### 2. 对香港"国际金融中心"地位的评价

作者认为香港是：
- 国际金融资本从金融领域侵略中国的"桥头堡"
- 中国官僚买办金融资本勾结国际金融资本的场所
- 向国际金融资本奉送货币主权、金融主权、经济主权的"罪恶之地"

### 3. 作者认为的影响

| 影响 | 描述 |
|------|------|
| 国际金融寄生 | 国际金融资本对中国经济的寄生更加牢固 |
| 美元霸权强化 | 美元霸权对中国经济的寄生关系得以延续 |
| 去美元化受阻 | 中国乃至全世界的去美元化斗争遭遇挫折 |
| 监管架空 | 中国对虚拟货币"国内严禁、国际严管"的政策被架空 |

### 4. 对跨国银行的批评

作者认为汇丰、渣打等跨国银行：
- 在中国享有"治外法权"
- 不受中国法律管辖
- 是缅北诈骗、金融犯罪资金转移出境的帮凶

## 作者的建议

- 收回两个"稳定币"牌照
- 驱逐汇丰银行和渣打银行
- 解散香港金融管理机构
- 废除港币，由中国中央政府指定国有银行负责货币发行
