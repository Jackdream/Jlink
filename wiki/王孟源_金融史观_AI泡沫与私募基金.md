# 王孟源：AI泡沫与私募基金的会计魔术

## 核心论点

作者将 **AI** 和 **Private Equity（私募基金）** 列为当前最离谱的两大泡沫焦点，并揭示它们之间的互相勾连。

## 主要内容

### AI泡沫特征

- AI估值虚高，存在大量"无中生有"的资金流动
- OpenAI与Microsoft、Oracle、NVIDIA、AMD等公司的左手倒右手游戏

### 私募基金新动作

2026年4月，路透社独家报道：

| 项目 | 内容 |
|------|------|
| 参与方 | OpenAI + 四家超大型私募基金 |
| 操作方式 | 强迫被投企业以浮夸价格购买AI服务 |
| 回报承诺 | **每年17.5%的投资回报率**（基于总投资额） |
| 对比 | Madoff诈骗案仅承诺8-10% |

### 会计魔术机制

```
私募基金账目凭空多出巨额获利
        ↓
OpenAI获得巨额营收
        ↓
为IPO中更离谱的估值正名
```

## 作者预判

作者早在2026年3月14日就预判AI和私募基金会互相勾连，两天后路透社即证实这一判断。
