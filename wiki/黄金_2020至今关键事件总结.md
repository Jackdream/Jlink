# 2020-2026黄金关键事件总结

## 一、2020年：疫情冲击与避险需求

### 关键事件
| 时间 | 事件 |
|------|------|
| 2020年3月 | 美股多次熔断，现货黄金跌破1880美元 |
| 2020年12月 | 中国外汇储备32165亿美元，黄金储备1948吨 |
| 2020年 | 全球央行净买入黄金326吨，连续11年净买入 |

### 特点
- 疫情引发流动性危机，股市暴跌
- 美债收益率降至历史低位
- 黄金作为避险资产受到关注

## 二、2021年：通胀与加息预期

### 关键事件
| 时间 | 事件 |

| 2021年1月 | 耶伦放弃弱美元政策，市场预期回归强美元 |

| 2021年5月 | 黄金突破1800美元关口 |

- 美债收益率大幅上行施压黄金
- 金价在1700-1900区间震荡

## 核心驱动因素

1. **美元信用弱化**：美债规模膨胀，美元避险功能下降
2. **地缘政治风险**：俄乌冲突、中东局势
3. **通胀压力**：全球高通胀环境下黄金保值功能凸显
4. **央行去美元化**：各国央行增持黄金替代美元储备
5. **美联储政策转向**：降息预期利好黄金
