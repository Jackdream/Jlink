# 中国海洋石油有限公司 2025年年报

## 公司概况

**中国海洋石油有限公司**（CNOOC Limited）是中国最大之海上原油及天然气生产商，亦为全球最大之独立油气勘探及生产企业之一。

| 董事长 | 张传江 |
| 注册地址 | 香港花园道1号中银大厦65层 |

**核心业务**：勘探、开发、生产及销售原油和天然气

**资产分布**：以中国海域的渤海、南海西部、南海东部和东海为核心区域，资产遍及亚洲、非洲、北美洲、南美洲、大洋洲和欧洲。

## 2025年主要成就

### 1. 储量创历史新高
- 净证实储量达 **77.7亿桶油当量**
- 首次荣获"全球最佳国家石油公司勘探企业"称号

### 2. 产量持续增长
- 油气净产量达 **777.3百万桶油当量**，同比增长 **7.0%**
- 天然气净产量达 1,037.3十亿立方英尺，同比增长 **11.6%**
- 连续多年保持增长

### 3. 勘探成果丰硕
- 全年获得 **6个新发现**，成功评价 **28个含油气构造**
- 中国海域：龙口25-1、曹妃甸22-3、锦州27-6、惠州21-8、涒洲10-5等
- 涒西南凹陷多类型潜山领域获重大突破
- 秦皇岛29-6 **亿吨级油田**
- 陆地新增天然气探明地质储量**超千亿方**

### 4. 项目建设加速
- **16个新项目**成功投产
- 在建项目超过80个
- 产能建设提速12%，示范项目提速26%

### 5. 成本优势巩固
- 桶油主要成本 **27.9美元/桶油当量**
- 保持行业领先成本竞争力

### 6. 股东回报
- 末期股息：**每股0.55港元**
- 中期股息：每股0.73港元
- **全年股息合计1.28港元**，股息支付率45%

## 区域布局

### 中国（占比69.1%产量）
| 区域 | 特点 |
|------|------|
| 渤海 | 中国最大原油生产基地，64.5%净证实储量 |
| 南海西部 | 重要天然气产区 |
| 南海东部 | 重要原油产区 |
| 东海 | 长三角地区天然气供应 |
| 陆地 | 非常规天然气（临兴、神府、潘河三大基地） |

- 海外资产占总油气资产 **40%**
- 项目分布于20多个国家：印尼、澳大利亚、尼日利亚、伊拉克、乌干达、阿根廷、美国、加拿大、英国、巴西、圭亚那、阿联酋等

- "深海一号"智慧气田

## 主要风险因素

1. **宏观经济风险**：地缘政治、全球货币政策影响
2. **油价波动风险**：OPEC政策、供需变化
3. **气候变化风险**：碳排放政策、能源转型
4. **地缘政治风险**：海外项目受制裁影响（Arctic LNG 2项目）
5. **HSSE风险**：安全环保
