# 全球主要金库体系

## 一、官方黄金托管核心枢纽

### 1. 纽约联邦储备银行金库
- **位置**：纽约曼哈顿金融区地下25米，建于1920年代
- **规模**：约6,331吨黄金（截至2024年），**全球最大黄金金库**
- **特点**：存储美国及其他国家/国际组织的黄金，布雷顿森林体系历史产物

### 2. 伦敦黄金存储体系
| 机构 | 规模/特点 |
|------|-----------|
| 英格兰银行金库 | 全球第二大金库，约5,000吨 |
| LBMA认证商业金库 | 汇丰、摩根大通、Brink's等6家机构，支撑68%现货黄金交易 |

- 截至2025年7月，伦敦金库黄金总持有量：8,865吨（价值约9,402亿美元）

### 3. 国际清算银行（BIS）托管体系
- **位置**：瑞士巴塞尔
- **规模**：管理约3,000多吨黄金，占全球央行黄金储备10%
- **服务**：为全球60个成员国央行提供黄金托管

### 4. 中国黄金存储体系
| 机构 | 特点 |
|------|------|
| 上海黄金交易所交割库 | 2023年场内交割量全球第一（3,912吨），超30家境外机构参与 |
| 深圳前海保税金库 | 东南亚国家新选择，如柬埔寨存入54吨黄金 |

- 中国黄金储备：2,262.39吨（2024年），连续15个月增持

### 5. 巴黎法兰西银行金库
- 法国全部2,437吨黄金储备（世界第四）存放于巴黎
- 2025-2026年出售最后129吨存放在纽约的黄金，完成"黄金回家"战略

## 二、新兴区域黄金枢纽

### 1. 迪拜金库体系
- 由迪拜多种商品交易中心（DMCC）运营，中东最大非主权黄金储存库
- 占全球黄金贸易15%，主要服务亚洲与欧洲之间黄金贸易
- 阿联酋2023年出口价值468亿美元黄金，世界第二大黄金出口国

### 2. 新加坡金库
- "The Reserve"等设施2025年黄金和白银存储订单同比增长88%
- 可储存高达500吨黄金（约640亿美元）
- 凭借政治稳定与税收优惠，正成为新兴黄金避险中心

### 3. COMEX黄金交割仓库（纽约）
| 机构           | 占比              |
| ------------ | --------------- |
| Brink's Co金库 | 核心组成部分          |
| 汇丰银行和摩根大通金库  | 管理82%以上COMEX总库存 |
| 丰业银行金库       | 可交割比例不足10%      |

## 三、黄金存储模式趋势

### 1. 境内存放模式
| 国家 | 特点 |
|------|------|
| 中国 | 绝大多数在国内，少量存于BIS |
| 俄罗斯 | 全部存于莫斯科和圣彼得堡 |
| 美国 | 全部存于国内 |

### 2. 分散存放模式
- **德国**：2013-2017年从纽约和巴黎运回674吨，但仍保留部分海外存储
- **荷兰**：38%存于加拿大，31%在国内，31%在纽约

### 3. "黄金回家"趋势
- **印度**：2025年将超过65%黄金储备转移至国内
- **东南亚**：柬埔寨、老挝、泰国将黄金从伦敦转移到上海
- **驱动因素**：对西方金融体系稳定性担忧，特别是俄乌冲突后西方冻结俄罗斯资产

---

## 关键洞察

> 全球黄金存储格局正从单极（西方中心）向多极转变，越来越多国家将黄金从伦敦、纽约、苏黎世转移到本国或新兴枢纽（上海、新加坡、迪拜），以增强金融自主性和抵御制裁风险。
