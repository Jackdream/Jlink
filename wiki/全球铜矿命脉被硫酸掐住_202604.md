# 全球铜矿命脉被硫酸掐住

> 来源：扑克财经
> 发布：2026年4月13日 20:30

---

## 核心观点

> **铜市场正从"资源约束"切换到"化学约束"**
> 
> 过去我们问：哪里还有矿？现在我们必须问：哪里还有酸？

---

## 全球铜供应链新逻辑

### 传统链条

```
矿山开采 → 精矿冶炼 → 电解铜
```

### 新链条（被替代）

```
硫磺 → 硫酸 → 湿法铜 → 全球供应
```

> 这条链条的两端，正被两股力量同时撕扯：
> - 东方大国的一纸禁令
> - 中东海峡的硝烟战火

---

## 危机一：东方"关闸"

### 中国硫酸出口限制

| 指标 | 数值 |
|------|------|
| 限制生效时间 | 2026年5月 |
| 2025年中国硫酸出口量 | 约460万吨 |
| 其中流向智利比例 | 约30% |
| 流向智利绝对量 | 超100万吨 |
| 前两月"酸路"萎缩 | 已萎缩50% |

### 根本原因

| 需求方 | 硫酸用途 |
|--------|----------|
| 新能源电池 | 电池材料生产 |
| 磷肥 | 农业化肥 |
| 高端制造 | 工业生产 |

> **中国自身对硫酸的需求正在膨胀。当内部需求与外部供应发生冲突时，天平毫不犹豫地倾向了前者。**

### 对智利的影响

- **智利**：全球最大铜生产国
- **依赖度**：每年从中国进口超100万吨硫酸
- **影响范围**：维持其**五分之一**的铜产量
- **工艺依赖**：氧化铜矿转化为可溶性铜液的唯一钥匙

---

## 危机二：中东"断源"

### 硫磺供应链

| 指标 | 数值 |
|------|------|
| 中东硫磺产量占全球 | 约25% |
| 经霍尔木兹海峡运输比例 | 约50% |
| 硫磺价格（中国港口） | 4750元/吨 → **6750元/吨** |
| 涨幅 | 约42% |

### 硫磺的不可替代性

> **硫磺没有替代品。**
> 
> - 铜可以被铝替代
> - 石油可以被新能源替代
> - 硫酸的唯一大规模来源，就是硫磺燃烧

---

## 湿法铜（SX-EW工艺）的困境

### 工艺特点

| 特点 | 说明 |
|------|------|
| 全称 | SX-EW（溶剂萃取-电解沉积） |
| 优势 | 成本低、投产快 |
| 原理 | 把硫酸洒在矿石堆上，等待含铜溶液渗出 |
| 角色 | 铜价上涨时最敏捷的响应者 |

### 当前困境

> **没有酸，就没有铜。这个公式简单到不需要任何经济学模型来验证。**

---

## 中国铜冶炼厂利润结构

### 加工费

| 指标 | 状态 |
|------|------|
| 铜精矿加工费 | **历史性进入负值区间** |
| 过去两年 | 几乎不赚钱 |

### 利润来源逆转

| 时期 | 利润结构 |
|------|----------|
| 过去 | 铜加工为主，硫酸为辅 |
| 现在 | **硫酸贡献超60%非主业利润** |

> 冶炼厂实质已经变成了"制酸厂"，生产铜只是顺带完成的任务。

### 两头夹击

| 方向 | 压力 |
|------|------|
| 上游 | 硫磺价格因中东战火而飞涨 |
| 下游 | 出口因政策限制而受阻 |

---

## 风暴中心

### 智利

智利是全球铜矿王冠，但底座建在进口硫酸的脆弱地基上。

| 公司/矿山 | 受影响情况 |
|-----------|------------|
| Antofagasta | 氧化矿需通过硫酸浸出 |
| Freeport (El Abra) | 硫酸供应风险写入年报重大事项 |
| BHP (Escondida) | 氧化矿堆浸工段无法独善其身 |

**智利硫酸价格过去1个月大涨44%**

### 非洲（刚果/赞比亚）

| 国家 | 应对措施 |
|------|----------|
| 赞比亚 | 2025年9月限制硫酸出口以保障本土供应 |

> 这种防御性动作本身，就是供应链极度紧张的最诚实信号。

---

## 全球硫酸供应链图谱

```
中东炼油厂（脱硫）
       ↓
硫磺运输（霍尔木兹海峡）
       ↓
中国港口 + 化工厂
       ↓
硫酸出口 → 智利矿山（湿法铜）
       ↓
全球铜供应
```

> 一条看不见的线，把波斯湾的油轮、中国港口的硫磺堆场、智利沙漠的堆浸场、刚果河畔的湿法槽、北美平原的磷肥厂，串成了一条首尾相衔的因果链。

## 核心结论

### 供应链弹性丧失

| 约束类型 | 说明 |
|----------|------|
| 过去 | 资源约束（哪里还有矿？） |
| 现在 | **化学约束（哪里还有酸？）** |

### 关键变化

| 过去 | 现在 |
|------|------|
| 铜的瓶颈在矿山 | 铜的瓶颈在化工厂的球形储罐和油轮的散装货舱 |

### 新问题

> **不再只是问：哪座矿山何时投产？哪条产线何时达产？**
> 
> **而是问：酸从哪里来？硫往哪里去？**
